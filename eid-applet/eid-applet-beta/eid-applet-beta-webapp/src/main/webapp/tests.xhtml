eID Applet Tests

Via this page you can test the eID Applet within your web browser.

eID Applet Identification This test will perform an eID identification using the eID Applet. A simple eID data readout will take place.

Run the eID Applet Identification test.

eID Applet Authentication This test will perform an eID authentication using the eID Applet. This test requires you to enter your eID PIN code.

Run the eID Applet Authentication test.
